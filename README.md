## Red Tools

Set of my masterpiece web tools; Meant to be very simple to manage.

---

Contain these binaries:

- atomic-merger
- atomic-scanner
- gen-hestia-node
- mysql-generator
- nginx-manager
- port-scanner

---

Use `make` for dropping them to `make uninstall` to revert them.

There's also .deb releases time to time.
